# Adaptive-Path-Planningfor-AUVs-in-Dynamic-Aquaculture-via-DDPG-GAN-Synergy
This work proposes a DDPG-GAN framework for AUV path planning in dynamic aquaculture. It integrates GAN-simulated hydrodynamics with DDPG trajectory optimization. Results show a 22.7% higher success rate and 37.0% better obstacle avoidance than baseline DDPG, ensuring robust autonomy in complex environments.
